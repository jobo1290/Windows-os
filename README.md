# GitHub Pages Deployment

This folder is ready to publish as a static site with GitHub Pages.

## Entry point

- `index.html` redirects to `Windows.html`, so the site can be opened from the repository root URL.

## Publish steps

1. Create a new GitHub repository.
2. Push the contents of this folder to the repository.
3. In the repository settings, open `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and the `/(root)` folder.
6. Save and wait for GitHub to publish the site.

The final address will usually be:

`https://<your-github-username>.github.io/<your-repository-name>/`
